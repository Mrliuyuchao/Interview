# **汉明距离**
### 主要思路
    开始吧两个数进行异或运算两个数中不同的数为1,
    在进行与运算的小技巧,n&(n-1)把为1的值去除,并且计数,
    最后得出来的就是两个数不同的数有多少个

####
    # 临时变量等于把输入的;两个数做异或运算 相同为0不同为一
    # 计数
    # 循环临时变量不等于零
        # 临时变量等于临时变量与临时变量减一做与运算  去除最后一个1 
        # 计数加等于1
        # 返回计数的值就是不同的个数
    
    
    def hammingWeight(x,y):
        a = x^y
        count = 0
        while a!=0:
            a = a&(a-1)
             count+=1
        return count
